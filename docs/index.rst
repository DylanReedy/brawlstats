Welcome to BrawlStats's documentation!
======================================

BrawlStats is an easy to use async wrapper for the unofficial `Brawl Stars API`_

Installation
~~~~~~~~~~~~

Install the latest stable version by typing this into your console:

::

   pip install brawlstats
